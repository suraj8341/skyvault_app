Skyvault — Weather Dashboard

A clean, real-time weather dashboard built with vanilla HTML, CSS, and JavaScript. Search any city to get current conditions, detailed stats, and a 5-day forecast — all in a sleek dark UI.

![Weather Dashboard](https://img.shields.io/badge/status-active-brightgreen) ![HTML](https://img.shields.io/badge/HTML-5-orange) ![CSS](https://img.shields.io/badge/CSS-3-blue) ![JavaScript](https://img.shields.io/badge/JavaScript-ES6+-yellow)

## ✨ Features

- 🔍 **City Search** — Search any city in the world instantly
- 🌡️ **Current Conditions** — Temperature, feels-like, weather description
- 📊 **Detailed Stats** — Humidity, wind speed, pressure, visibility
- 📅 **5-Day Forecast** — Daily highs and lows at a glance
- 🎨 **Polished Dark UI** — Smooth animations, ambient glows, responsive layout
- ⌨️ **Keyboard Friendly** — Press Enter to search

## 🚀 Getting Started

### 1. Clone the repo
```bash
git clone https://github.com/YOUR_USERNAME/skyvault-weather.git
cd skyvault-weather
```

### 2. Get a free API key
Sign up at [openweathermap.org](https://openweathermap.org/api) — the free tier is plenty.

### 3. Add your API key
In `index.html`, find this line and replace the placeholder:
```js
const API_KEY = 'YOUR_API_KEY'; // ← paste your key here
```

### 4. Open in browser
Just open `index.html` — no build step needed!
